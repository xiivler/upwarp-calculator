Upwarp Calculator

by Xiivler

This tool can be used to determine the greatest possible upwarp, or whether a specific upwarp is possible. If Mario has a negative vertical velocity, greater upwarps may be possible than what this tool reports. If you are trying to actually TAS a specific upwarp, it's best to use the "calculate if upwarp is possible" option, but the "calculate max upwarp" option is useful for strategizing.
